Site Web pour un restaurant

J'ai réalisé ce projet pour un client travaillant dans le domaine de la restauration.

L'objectif ? Permettre à ce restaurant de fafner en visibilité de et proposer son menu de pizzas à ces utilisateurs.

Mon client, novice dans les nouvelles technologies, a la possibilité de modifier facilement son menu en ligne grâce à une interface d'administration réalisée sur mesure.

Ce projet simple mais fonctionnel a donné entière satisfaction à mon client.

Technologies utilisées : HTML / CSS / DANGO / PYTHON / HEROKU
